# Power BI Data Analytics Project

## Overview
This project analyzes a B2B sales pipeline dataset for a computer hardware company. It focuses on comparing sales team performance, identifying lagging agents, investigating quarter-over-quarter trends, and evaluating product win rates.

## Repository Contents
- `Power BI Project.pbix` — Power BI report with the data model, transformations, and visuals.
- `Project Report.pdf` — Exported report/documentation for sharing insights.
- `Assignment.txt` — Original analysis prompts and questions.

## Analysis Questions
- How is each sales team performing compared to the rest?
- Are any sales agents lagging behind?
- Are there any quarter-over-quarter trends?
- Do any products have better win rates?

## How to Use
1. Open `Power BI Project.pbix` in Power BI Desktop.
2. Refresh data connections if prompted.
3. Review the report pages for insights and visuals.
4. Refer to `Project Report.pdf` for a shareable summary.

## Notes
If you are viewing this repository without Power BI Desktop, use the PDF report for a static view of the findings.
